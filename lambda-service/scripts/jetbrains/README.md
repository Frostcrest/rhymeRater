Copy `External Tools.xml` to the following location, you're ready for Serverless!  

### Windows
 
c:\Users\<USER_NAME>\.<PRODUCT><VERSION>\

### Linux
 
~/.<PRODUCT><VERSION>

### Mac

~/Library/Preferences/<PRODUCT><VERSION>
